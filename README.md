# Subjoin
:smiley:

![image](https://i.pinimg.com/originals/f9/09/35/f90935f6acfe8bb0170bc7b34020a465.png)

I constantly found myself in the process of creating/deleting reddit
accounts, and trying to figure out what dope subreddit(s) I was subscribed to,
so I created this program to join the different subreddits I was subscribed to
on other accounts.
